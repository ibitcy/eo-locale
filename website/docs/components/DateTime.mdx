`DateTime` component provides ability to format values of `Date`.
Full list of supported properties you can find on [MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/DateTimeFormat) page

To use this component, you need to import the component as the followings:

#### React

```jsx
import { DateTime } from '@eo-locale/react';
```

#### Preact

```jsx
import { DateTime } from '@eo-locale/preact';
```

#### React Native

```jsx
import { DateTime } from '@eo-locale/react-native';
```

And add into your application like this:

```jsx
export function SomeComponent() {
  return (
    <DateTime
      language={appLanguage}
      value={new Date()}
      day='numeric'
      month='long'
      weekday='long'
      year='numeric'
    />
  );
}
```

As above example, component will display <LanguageSelector><Highlight><Preview/></Highlight></LanguageSelector>
